 Project

How to Create Project?

Go to Dashboard page. Click the + button. Enter project detail. Click 'Create'. All detail must be filled in except the Share To column. This column allows you to specify who have the right to view this project. Enter the registered email to the column. Use ',' to seperate the email even only one email.

 Product Backlog

How to Create Product Backlog?

Select a project. Click on project's ID. Look at the Product Backlog Section. If no any PB item exists, then click on the 'Create Product Backlog' button. Enter the PB detail, click create. If PB item exists, click on the 'View Product Backlog' button. Redirected to PB page. Click 'Add Item' button. Enter PB detail, click create.

Sprint Backlog

How to Create Sprint Backlog?

Select a project. Click on project's ID. Look at the Sprint Backlog Section. If no any SB item exists, then click on the 'Create Sprint Backlog' button. Enter the SB detail, click create. It will automactically generate a Sprint Backlog. The created SB item will include in it. If SB item exists, Select a Sprint Backlog. Redirected to sB page. Click 'Add Item' button. Enter SB detail, click create.

Please notice that if SB ID changed, it will actomatically move the current item into another or new Sprint Backlog.
SB ID = Sprint Backlog ID; SB ITEM ID = Item ID inside a Sprint Backlog
Requirement Traceability Matrix

Select a project > Scroll Down the Project Detail Page > Select Generate RTM

Keyword and Backlog type must be filled. The generated RTM can be export to CSV file.
